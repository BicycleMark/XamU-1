Before you begin

This is an iOS class. To complete the coding exercises, you will need an iOS development environment with either a simulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This course introduces the most common user interface navigation metaphors for iOS. It also examines the most appropriate uses for each and how to implement them in Xamarin iOS apps.

Objectives

  1. Progress through pages of data with stack-based navigation
  2. Show different views of related data with tab navigation
  3. Display hierarchical relationships with master/detail navigation
