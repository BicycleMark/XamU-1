Master-detail and Drawer Navigation in Xamarin.Forms (XAM290)

In this Xamarin.Forms class, we discuss how to navigate between screens of content using drawer navigation and learn how to browse a collection of data using a Master-detail UI.

Upon completion you will be able to:

  • Choose between split and popover for your master view
  • Switch between pages using popover drawer navigation
  • Display a collection using master-detail split view

Exercises

  1. Exercise 1: Control the Master behavior of a MasterDetailPage

    In this exercise, you will influence how a MasterDetailPage presents the master UI for drawer and master-detail UI.

  2. Exercise 2: Define the UI for drawer navigation

    In this exercise, you will use a MasterDetailPage to create the UI required for drawer navigation.

  3. Exercise 3: Respond to drawer-menu selection to update your UI

    In this exercise, you will respond to selections made by the user on the drawer and update the presented details content.

  4. Exercise 4: Display a master-detail split view of a collection

    In this exercise, you will use a MasterDetailPage to display and navigate a homegenious collection of data with a list/detail side-by-side presentation.
